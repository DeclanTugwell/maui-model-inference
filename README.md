# Setup

Install appropriate .net 11 version for maui usage

Download ONNX model file from [hugging face](https://huggingface.co/onnx-community/gemma-3-1b-it-ONNX/resolve/main/onnx/model_q4f16.onnx_data)

Place the file in `OnnxExperimentation.Maui\Platforms\Android\Assets\Models\gemma-3-1b`

Run on android
